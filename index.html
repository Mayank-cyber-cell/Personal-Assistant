Voice Commands

Try these commands to interact with your assistant

"Alexa play [song name]" Plays song on YouTube
"Alexa time" Tells current time
"Alexa who is [person]" Wikipedia summary
"Alexa weather" Current weather info
"Alexa news" Top 3 news headlines
"Alexa joke" Random tech joke
"Alexa question" Ask general questions

Quick Setup

1. Clone Repository

git clone https://github.com/Mayank-cyber-cell/Personal-Assistant.git cd Personal-Assistant

2. Install Dependencies

pip install -r requirements.txt

3. Configure API Keys

Create a .env file with your API keys:

WEATHER_API_KEY=your_openweather_key NEWS_API_KEY=your_news_api_key IPINFO_TOKEN=your_ipinfo_token HUGGINGFACE_TOKEN=your_huggingface_token

4. Run the Assistant

python main.py
